# SSH Warning MOTD

Auto-installs SSH warning banners with IP/location detection on any server.

## Supported OS
- Proxmox VE
- Ubuntu / Debian
- CentOS / RHEL
- Oracle Linux

## Install (one command)
```bash
curl -s https://raw.githubusercontent.com/DanielG158/motd/main/vm-motd-installer.sh | sudo bash
```
